[Talks](https://thib.me/categories/talks/)
=====

Public talks in either french or english, given at various events. All tallks are available at https://thib.me/categories/talks/.

## Deployment

```sh
./deploy.sh
```

For the first time, do this first:

```sh
mkdir build
cd build/
git init
cd ..
```
