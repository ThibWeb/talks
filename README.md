[Talks](http://talks.thib.me)
=====

Public talks in either french or english, given at various events. All tallks are available on [http://talks.thib.me/](http://talks.thib.me/).

## Deployment

```sh
mkdir build
cd build/
git init
cd ..
./deploy.sh
```
